# coordinate

Simple coordinate picker based on OpenStreetMap and Leaflet.

## Features

- Click map to get coordinates
- Browser geolocation support
- Convert between `BD-09`, `GCJ-02`, and `WGS84`
- Copy longitude, latitude, or full coordinate pair

## Local preview

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
